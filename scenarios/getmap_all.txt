<p>The aim of this test is to measure the response time of the server for the <b>GetMap</b> request with vector and raster layers. The extent is based on the most extended layer.</p>

<p>Dataset description:</p>

<ul>
  <li> 1 Point layer named <i>adress_ban</i> (~ 450 000 features) with the <i>Single symbol</i> renderer and without labels
  <li> 1 MultiLineString layer named <i>hydro_course.geomsimple</i> (~ 4 500 features)
  <li> 1 Line layer named <i>hydro_segment</i> (~ 14 000 features) with the <i>Categorized</i> renderer and without labels
  <li> 1 Polygon layer named <i>hydro_bassin</i> (6 features) with the <i>Single symbol</i> renderer and with <i>Single labels</i>
  <li> 1 Polygon layer named <i>vw_clc2006</i> (~ 3 000 features) with the <i>Categorized</i> renderer and without labels
  <li> 1 Polygon layer named <i>hydro_lake</i> (~ 800 features) with the <i>Single symbol</i> renderer and without labels
  <li> 1 Polygon layer named <i>landuse_corine_lc_2006</i> (~ 3 000 features) with the <i>Categorized</i> renderer and without labels
  <li> 1 Raster layer named <i>eu_dem_v11</i> (5324x3374)
</ul>
