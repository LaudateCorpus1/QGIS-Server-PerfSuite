<p>The aim of this test is to measure the response time of the server for the <b>GetMap</b> request with vector layers based on lines geometries.</p>

<p>Dataset:</p>

<ul>
  <li> 1 MultiLineString layer named <i>hydro_course.geomsimple</i> (~ 4 500 features)
  <li> 1 Line layer named <i>hydro_segment</i> (~ 14 000 features) with the <i>Categorized</i> renderer and without labels
</ul>
